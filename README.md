# Game of Life in [Ada](https://www.adaic.org/)

![thumbnail](thumbnail.png)

Simple Game of Life implementation in Ada.

## Quick Code

Install [GNAT](https://www.adacore.com/download) first.

```console
$ gnatmake -gnatwa gol.adb
$ ./gol
```
